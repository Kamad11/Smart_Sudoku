# Smart Soduko
## 3 modes to play
* Real time
* Upload
* Random generator

### Real Time mode
* Capture frame
* Preprocessing - Gaussian blur, Adaptive threshold
* Find contours
* Find biggest contour
* Recognize digits
* Display solution
* Increase the speed of the process

### Upload Image
* Read image
* Preprocessing - Gaussian blur, Adaptive threshold
* Find contours
* Find biggest contour
* Recognize digits
* Display solution
